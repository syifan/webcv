Embed EasyCV without a framework

This page uses only a couple of CDN imports, a YAML file, and a few lines of JavaScript. Drop it into any static site to render your resume with EasyCV.

1

Keep cv_data.yml in the same folder as this HTML.

2

Serve the folder locally (e.g. python3 -m http.server 4173).

3

Open the page and EasyCV renders itself—no build tools required.

How it works: EasyCV and js-yaml are imported directly from CDNs, the YAML is fetched, and renderCv mounts the page inside #cv-root.
